BashScript
==========

RoadMap of BashScript Project:  

1) DevSecOps Scripts on Linux Platform for Java Projects.  
(version 1.0)  

- /src/Template/RillHost-BashScript-Template.sh   define the template of the script.
- /src/Toolbox/RillSoft-BashScript-All-Zero.sh    define the general loading program of the toolbox scripts.   
- /doc/<?!.md>                                       define the BashScript Project Documents.

2) DevSecOps Scripts on Linux Platform for Data Projects. (version 2.0)  
3) DevSecOps Scripts on Linux Platform for Cash (Coin) Projects (version 3.0). - Financial Projects
